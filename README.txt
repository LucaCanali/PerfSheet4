
Title: PerfSheet4 is a tool aimed at DBAs and Oracle performance analysts. It provides a simplified interface to extract and visualize Oracle AWR data using Excel pivot charts.

Version: V3.7.1, February 2015 (the first PerfSheet4 release was in March 2013)

Authors: Luca.Canali@cern.ch from an original idea and PerfSheet 3 by Tanel@TanelPoder.com

Contents: 

- PerfSheet4_v3.7.xlsm -> PerfSheet4, run under Excel 2010 or 2013.
- PerfSheet4_v3.7_noActivex_controls.xlsm -> PerfSheet4 modified by removing Activex Controls, this should run also on older versions of Excel and/or if there are compatility problems with Excel Activex Controls
- sqlplus_scripts -> directory of selected scripts to extract data from AWR
